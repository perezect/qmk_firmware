# Generic Corne Layout by perezect

Currently only (cleanly) supports a windows computer.

Supports OLED screens and layer-specific LED configurations

Leverages the RGB_MATRIX over RGBLIGHT at the moment, although there is some unmaintained RGBLIGHT
code present.

[Leverages the Luna Pet by HellSingCoder](https://github.com/HellSingCoder/qmk_firmware/tree/master/keyboards/sofle/keymaps/helltm)
